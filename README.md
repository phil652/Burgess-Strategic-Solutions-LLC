# Burgess Strategic Solutions Website

Static website files for Burgess Strategic Solutions LLC.

## Upload to GitHub

Upload everything in this folder to the root of the new repository:

- `index.html`
- all other `.html` pages
- the `assets/` folder
- `.nojekyll`

For GitHub Pages, use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

The home page is `index.html`.
