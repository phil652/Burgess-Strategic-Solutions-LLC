# Upload These Files to GitHub

Upload the contents of this folder to the root of your GitHub repository.

The repository root should show:

- `index.html`
- `about.html`
- `services.html`
- the other `.html` pages
- an `assets` folder
- `.nojekyll`

Important:

- Do not upload only the `.html` files. The `assets` folder is required for the logo, styling, mobile menu, and hero image.
- Do not upload this folder as a folder inside the repository. The files inside this folder need to be at the repository root.
- In GitHub Pages settings, choose `Deploy from a branch`, branch `main`, folder `/root`.
